Mr. Jeff B Edwards (Goodsoftware) bio
Jeff Edwards has been a software developer for 21 years. mainly working with 4GL's (Omnis, 4D, webobjects), and for the last 3 years with open source technologies and specifically python as the language utilised by application framework for complex database applications deployed on the browser. Jeff runs several software companies and has developed database applications for enterprise size businesses all over Australia. He is strategic partner for Softwell who developed the Genropy framework, an open source RAD environment for building powerful database applications deployed on the web.
